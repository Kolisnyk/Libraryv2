
Добрый день.

В Проэкте реализован следующий функционал:
**Зарегистрированный пользователь может добавить книгу как в общую так и библиотеку черновиков;
**Любой пользователь может только читать перечень книг в общей бибиотеке;
**У любой книги есть название; автор; возможность вставлять ссылки на превью обложки (изображение),
  т.к. база данных станет "тяжелой" если в нее складывать картинки;краткое содержание и категория (жанр), 
  одна книга может быть одновременно в нескольких жанрах;
**В библиотеке черновиков книги может просматривать только зарегестрированный пользователь;
**Встроен справочник параметров, по которым может осуществляться выдача книг из библиотеки.


Особенность Проэкта:
**Реализация Front-end не произведена, т.к. программа набиралась в RubyMine, и руками не 
нарисовал страничку;
**Любой пользователь может просматривать все книги за все время, просматривать заданное им 
количество последних записей или делать выборку по интересной ему тематике, по ID.

Работа с Проэктом.
Для работы с библиотекой необходимо следующее педустановленное ПО: Ruby, MYSQlite.
Меню реализовано на английском языке.В теле программы на самых нужных местах, по моему мнению ), стоят комментарии.
В случае, если нужно произвести запись или чтение книги в Библиотеку, запускается файл library.rb. 
Для чтения книг из библиотеки апускается файл read.rb. Для вызова help используется параметр -h.

Для использования полного функционала программы  прошу использовать логин 1111, пароль 1111.
